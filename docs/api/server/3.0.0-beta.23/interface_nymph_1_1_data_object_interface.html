Nymph\DataObjectInterface::refresh ( )

Refresh the object from storage.

(Bypasses Nymph's cache.)

If the object has been deleted from storage, the database cannot be reached, or a database error occurs, refresh() will return 0.

Returns
bool|int False if the data has not been saved, 0 if it can't be refreshed, true on success.
